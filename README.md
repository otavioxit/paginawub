# Futebol Feminino: Origem e Resistência ⚽⚽️

Este é um projeto web minimalista e totalmente responsivo que conta a história, os desafios e a evolução do futebol feminino no mundo e no Brasil.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização moderna utilizando **Flexbox** e a função `clamp()` para total responsividade (de 320px a 2560px).
- **JavaScript**: Comportamento dinâmico utilizando boas práticas como `addEventListener`.

## 📦 Como Executar o Projeto

1. Baixe ou clone esta pasta no seu computador.
2. Abra o arquivo `index.html` em qualquer navegador de internet (Chrome, Edge, Firefox, etc.).
3. Clique no botão "Clique para Inspirar" no final da página para interagir.
